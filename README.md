# PromptArsenal
PromptArsenal是一个开源的prompts资源仓库,致力于收集和整理全网最全、最新、最高质量的文本生成prompts资源。用户可以在PromptArsenal找到各种类型的prompts,包括基于规则的prompts、基于数据的prompts以及最新研究成果prompts等。PromptArsenal的资源覆盖了英文和中文,涉及各个领域,满足用户多样化的需求。
